Destructible Trees

While I did not code the destruction of the trees, I did create the models and textures for them. This required me to set things up to easily be broken apart. The models themselves while looking detailed, are fairly low poly. This allows for large scale maps that utilize heavy vegetation to run fairly well.

This was recorded in a unfinished side project me and a friend are working on. Will add more relating to this project eventually.
